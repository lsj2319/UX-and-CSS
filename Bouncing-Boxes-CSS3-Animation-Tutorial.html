Bouncing with CSS Tutorial

In the first tutorial of this series we got our feet wet with CSS animations by changing the colors of two divs. This time we learn how to move objects and do a little text manipulation. That's right - moving objects through space using only CSS3 and HTML.

We will a couple create a couple of objects and bounce them.

You will need:

  • A modern browser (IE 10 +, Recent versions of Firefox, Google Chrome, or Safari, other browers your milage may vary)
  • A text editor like Notepad or JSFiddle.net or Sublime
  • A sense of adventure

 

The HTML

I've created this fork of a tutorial by Honkiat. I made two divs with two different classes. One box bounces slowly and the other bounces quickly. Last week we talked a little bit about percentages, and as you look at the CSS code you will see the timing of the bouncing is controlled using percentages. By changing those, we change the appearance of the bounce effect.  You can grab the code or see it in action on JSFiddle.net

Paste this HTML in between the body tags of an HTML file, or in your JSFiddle:
<div class="css3-bouncer">
<p>This box bounces!!!</p>
</div>
<div class="css3-bouncer2">
<p>This box bounces too!</p>
</div>
<p>
Based on a tutorial by Honkiat (http://www.hongkiat.com/blog/css3-bounce-effect/)
</p>

What Can We Animate?

What kinds of things could we animate? The specifications for the CSS Animation property on MDN (Mozilla Developers Network) or www.W3C.org reveal that the primary things we can animate are colors and text.

We can easily animate properties of an element such as:

  • color (text)
  • background-color (block elements)
  • height
  • width

Transforming elements by scaling and rotating them can be very useful. This also does not tax performance the way a lot of other animations can. Transform can be great for animating charts so they grow in different ways. Use the following properties:

  • transform: scale()
  • transform: rotate()
  • transform: translate()
